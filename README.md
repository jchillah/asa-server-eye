# ASA Server Eye

ARK Survival Ascended companion app showing real-time server population.

## Features (MVP)

- Server list
- Server details
- Favorites
- Settings
- Localization (English / German)

## Planned Features

- Player sightings
- Push notifications
- Watchlists
- Premium alerts

## Tech Stack

Flutter  
Riverpod  
GoRouter  
Clean Architecture  
Feature-First Structure  

## Architecture
lib
├── app
├── core
├── features
│ ├── servers
│ ├── favorites
│ └── settings
└── l10n


## Development Workflow

1. Create issue
2. Create feature branch
3. Implement feature
4. Open pull request
5. Merge to main
